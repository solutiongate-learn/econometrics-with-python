# Data Folder

This folder contains cleaned datasets used across the notebooks.

## trinidad_tobago_core.csv

Core macroeconomic time series data for Trinidad and Tobago, based on the examples in Watson & Teelucksingh (2002).

**Period:** 1967–1991 (25 annual observations)

**Key Variables:**
- `Year`: Observation year
- `IMPORTS`: Imports of goods and services (million $)
- `INCOME`: Income / GDP (million $)
- `RATIO`: Relative price ratio (import prices / domestic prices)

**Source:** Derived from examples in Chapter 1 of the book (Appendix 1.2).

**Note:** This is a cleaned and structured version prepared for teaching purposes. The original full dataset is referenced in the book as `WT_DATA.XLS`.
